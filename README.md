# AI Search Agent

I built this as my first AI agent project. It's a simple chat assistant 
that can search Google in real time to answer your questions.

## What it does

You type a question in the chat, and the agent searches Google, 
reads the results, and gives you a smart answer. It also remembers 
what you said earlier in the conversation so it feels natural to talk to.

## Tools I used

- **n8n** — to build and connect everything together
- **OpenAI** — the AI brain that understands and responds
- **SerpApi** — to search Google in real time
- **n8n Memory** — so it remembers the conversation

## How to use it

1. Download the `` file
2. Import it into your n8n account
3. Add your OpenAI and SerpApi API keys
4. Hit activate and open the chat
5. Start asking questions!

## Why I built this

I wanted to learn how AI agents work in practice. 
This was my first project combining AI with real time web search 
and it honestly surprised me how powerful it turned out to be.
